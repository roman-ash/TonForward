TonForward - это Telegram-площадка, которая связывает пользователей, желающих приобрести товары из других городов или стран, с местными и путешествующими байерами, которые могут купить и переслать эти товары.

Для пользователей в России TonForward работает как обычный Web2-сервис: клиенты платят в рублях через классические платёжные системы, а сервис организует покупку и доставку товара через выбранного байера. Под капотом платформа использует смарт-контракты TON только во взаимодействии между сервисом и байером - для управления эскроу, выплатами и разрешения споров - без прямого контакта конечных пользователей с криптовалютой.

Сервис поддерживает собственный пул криптовалюты, за счёт которого обеспечивает каждое рублёвое поступление и из которого выполняются выплаты байерам по условиям смарт-контрактов.

Логика сделки в TonForward:

1. Заказчик создаёт заявку: что купить, где, за сколько и в какие сроки доставить.

2. Байер откликается, заказчик выбирает подходящего исполнителя.

3. Заказчик оплачивает заказ в рублях сервису (стоимость товара + вознаграждение байеру + комиссия сервиса).

4. Сервис конвертирует нужную сумму в TON и фондирует смарт-контракт эскроу между собой и байером.

5. Байер покупает товар за свои деньги, прикладывает фото и чек, затем отправляет товар заказчику и указывает трек-номер.

6. После получения посылки заказчик либо подтверждает доставку (эскроу выплачивает байеру вознаграждение и компенсирует покупку), либо открывает спор.

7. В случае спора сервис выступает арбитром и через смарт-контракт TON принимает решение - средства возвращаются сервису (для возврата заказчику) или перечисляются байеру в зависимости от результата разбирательства.

Stack (MVP):

Backend: Python, Django, Django REST Framework

DB: PostgreSQL

Async / cron: Celery

Blockchain: TON, Tact smart contract Deal (escrow between platform and buyer)

Integration: TonCenter API, tonsdk

Client: Telegram Mini App / web frontend (API-driven)
