# Устранение проблем при установке Tact

## Предупреждение EBADENGINE (Node.js версия)

Если вы видите предупреждение:
```
npm WARN EBADENGINE Unsupported engine {
  package: '@tact-lang/compiler@1.6.13',
  required: { node: '>=22.0.0' },
  current: { node: 'v20.11.1' }
}
```

### Это критично?

**НЕТ, это не критично!** Это просто предупреждение о том, что пакет рекомендует Node.js 22+, но обычно работает и на более старых версиях.

### Что делать?

1. **Сначала проверьте, работает ли Tact:**
   ```bash
   tact --version
   ```
   
   Если команда работает и показывает версию - все отлично, можно продолжать!

2. **Если Tact не работает или нужна совместимость:**
   
   Обновите Node.js до версии 22+:
   
   - **Windows/Linux/macOS:** Скачайте с https://nodejs.org/
   - Или через nvm (Node Version Manager):
     ```bash
     nvm install 22
     nvm use 22
     ```
   
   Но обычно обновление не требуется, если Tact уже работает.

### Предупреждение deprecated inflight

```
npm WARN deprecated inflight@1.0.6: This module is not supported...
```

Это тоже **не критично**. Это зависимость других пакетов, не ваша проблема. Игнорируйте это предупреждение.

## Проверка работоспособности

После установки проверьте:

```bash
# Проверка версии
tact --version

# Проверка помощи
tact --help

# Попытка компиляции (из директории contracts)
cd contracts
tact compile Deal.tact
```

Если все команды работают - можете игнорировать предупреждения!

## Частые проблемы

### Команда `tact` не найдена после установки

**Windows:**
- Перезапустите терминал/PowerShell
- Проверьте PATH: `echo %PATH%`
- Проверьте, где установлен: `npm config get prefix`

**Linux/macOS:**
- Перезапустите терминал
- Проверьте PATH: `echo $PATH`
- Добавьте путь npm в PATH если нужно:
  ```bash
  export PATH="$PATH:$(npm config get prefix)/bin"
  ```

### Ошибки при компиляции контракта

1. Проверьте синтаксис Tact контракта
2. Убедитесь, что находитесь в правильной директории
3. Проверьте версию Tact: `tact --version`
4. Попробуйте компилировать с флагом `--verbose`:
   ```bash
   tact compile Deal.tact --verbose
   ```
